# BlockState

The state of an block. Initially `ACCEPTED` and after auction result publication updated to either `EXECUTED` or `REJECTED`. 

## Enum

* `ACCEPTED` (value: `'ACCEPTED'`)

* `EXECUTED` (value: `'EXECUTED'`)

* `REJECTED` (value: `'REJECTED'`)
