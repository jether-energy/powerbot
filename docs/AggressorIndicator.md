# AggressorIndicator

Indicates whether the executed order was a trade aggressor or trade originator. * Y - Trade aggressor * N - Trade originator * U - Unknown, for executed orders of remote products and data before migration

## Enum

* `Y` (value: `'Y'`)

* `N` (value: `'N'`)

* `U` (value: `'U'`)
